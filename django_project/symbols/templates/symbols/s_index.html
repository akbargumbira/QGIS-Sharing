QGIS Symbol Repository

QGIS Symbol Respository provides a common place for sharing symbols. The symbols can be imported directly into QGIS Desktop application through the Import/Export option in the Symbol Manager.

To upload a new set of symbols go to Symbol Upload Form and upload the XML file exported by the QGIS Symbol Manager.
